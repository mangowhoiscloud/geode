# Dark Fantasy IP Analysis Rules

## Genre Characteristics
- High core fandom loyalty → increase Community Momentum(J,K,L) weight
- Adult target audience → Conversion metric(C1) importance is high
- Hardcore gamer affinity → stricter Core Mechanics(A) evaluation

## Special Considerations
- If the original has high violence/gore levels, reflect platform rating risk (Licensing Complexity)
- For long-running series (10+ years), increase Franchise Size weight
- For completed works, separately evaluate spin-off/prequel potential when assessing Expansion Potential(F)
