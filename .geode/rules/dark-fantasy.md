# 다크 판타지 IP 분석 규칙

## 장르 특성
- 높은 코어 팬덤 충성도 → Community Momentum(J,K,L) 가중치 상향
- 성인 타겟 → Conversion 지표(C1) 중요도 높음
- 하드코어 게이머 친화 → Core Mechanics(A) 평가 엄격화

## 특수 고려사항
- 원작 폭력/고어 수위가 높으면 플랫폼 등급 리스크 반영 (Licensing Complexity)
- 장기 연재물(10년+)은 Franchise Size 가중치 상향
- 완결작은 Expansion Potential(F) 평가 시 스핀오프/프리퀄 가능성 별도 검토
