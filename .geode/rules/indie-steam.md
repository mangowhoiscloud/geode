# Indie Steam Game IP Analysis Rules

## Genre Characteristics
- Already gamified IP → use ip_type=gamified path
- Steam performance data directly available (MonoLake DAU, Revenue)
- Organic community growth → Social Resonance(K) is important

## Special Considerations
- Multi-platform expansion is a key value driver
- Mobile porting potential → include Audience Crossover(P) evaluation
- Relationship with original developer → evaluate Licensing Complexity(T)
- If Steam review data is abundant, increase Review Score(C2) reliability
