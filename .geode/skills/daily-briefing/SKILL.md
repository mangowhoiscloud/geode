# Daily Briefing

Automatically summarizes AI/tech trends, job market updates, and news from areas of interest every day.

## Briefing Sections

### 1. AI/ML Trends
- Hugging Face trending models/papers
- Major announcements from Anthropic, OpenAI, Google
- News related to Agentic AI / MCP / LangGraph

### 2. Job Market
- AI/ML Engineer position trends
- Korea + Global (remote) job postings
- Salary/compensation trends

### 3. Areas of Interest
- YouTube creator/developer community trends
- Open-source agent framework updates

## Briefing Format

```markdown
## Daily Briefing — YYYY-MM-DD

### AI/ML
- [Headline] — Summary (source)

### Jobs
- [Company/Position] — Key requirements (source)

### Interest
- [Topic] — Summary (source)

---
Generated by: GEODE auto-briefing
```

## Schedule Integration

```
/schedule create "daily at 9:00" action="Generate today's AI/jobs/tech briefing"
```

When registered as a scheduled job, it runs automatically each morning in an isolated session.

## Guidelines

- Keep each section concise with 3-5 items
- Include only news from the last 24 hours
- Summarize in English, attach original source links
- After completion, save to project memory via memory_save
