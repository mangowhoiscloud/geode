# GEODE — Claude Code Project Memory

이 파일은 Claude Code 하네스의 프로젝트 메모리입니다.
GEODE 에이전트의 런타임 메모리는 `.geode/memory/PROJECT.md`에 있습니다.

## 프로젝트 구조
- `.claude/` — Claude Code 하네스 (settings, hooks, skills)
- `.geode/` — GEODE 에이전트 시스템 (memory, rules, journal, vault)
- `GEODE.md` — 에이전트 정체성 (Karpathy program.md 패턴)
- `CLAUDE.md` — 개발자 하네스 지시서

## 도메인 규칙
- `.geode/rules/` 디렉토리의 .md 파일이 에이전트에 자동 로딩됩니다
