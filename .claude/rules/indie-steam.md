# 인디 Steam 게임 IP 분석 규칙

## 장르 특성
- 이미 게임화된 IP → ip_type=gamified 경로 사용
- Steam 실적 데이터 직접 활용 가능 (MonoLake DAU, Revenue)
- 커뮤니티 자생적 성장 → Social Resonance(K) 중요

## 특수 고려사항
- 멀티플랫폼 확장 여부가 핵심 가치 드라이버
- 모바일 포팅 가능성 → Audience Crossover(P) 평가 포함
- 원작 개발사 관계 → Licensing Complexity(T) 평가
- Steam 리뷰 데이터가 풍부하면 Review Score(C2) 신뢰도 상향
